# Проект "Прогнозирование заказов такси" (status: In review)

## Описание проекта: 
Чтобы привлекать больше водителей в период пиковой нагрузки, нужно спрогнозировать количество заказов такси на следующий час.

## Цель проекта
Проанализировать имеющиеся данные и разработать модель, способную предсказать количество заказов на следующий час не больше, чем 48 по метрике RMSE.

## Стек технологий
catboost, LightGBM, statsmodels, sklearn, pandas, numpy, seaborn, matplotlib
